File Name Maker

String To CLI-Friendly File Name Converter

Use your browser console to pass 3 parameters(your non-CLI friendly fileName str, optional additional string add to end of fileName unless 3rd argument is 'a', and if optional 3rd argument is 'a' then the additional string argument is added to the beginning)
to the makeFileName() function
to get a CLI friendly file name.

Note: The makeFileName() function may not be sufficient for all desired file name outputs.
